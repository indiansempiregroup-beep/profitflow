# Color Palette

## Core Palette

- Primary: #2563EB
- Primary Dark: #1D4ED8
- Secondary: #0F172A
- Surface: #F8FAFC
- Border: #E2E8F0
- Text Primary: #0F172A
- Text Secondary: #475569
- Success: #16A34A
- Warning: #D97706
- Error: #DC2626

## Usage Rules

- Use primary blue for highlights and calls to action
- Use navy for high-value financial content
- Use green for profitable signals and amber for caution states
- Reserve red for destructive or critical states
