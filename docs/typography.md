# Typography System

## Type Scale

- Display: 32px / 700
- H1: 28px / 700
- H2: 24px / 600
- H3: 20px / 600
- Body: 16px / 400
- Caption: 12px / 400

## Usage Rules

- Use bold weights sparingly for emphasis
- Favor legible line heights for financial data
- Maintain consistent hierarchy across mobile and web surfaces
