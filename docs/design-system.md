# Design System

## Principles

- Premium, trustworthy, and calm
- Fast to scan and easy to act on
- Consistent hierarchy for financial data
- High contrast for readability under varied conditions

## Foundations

- Typography: Inter or SF Pro depending on platform
- Spacing: 4px base scale
- Color: deep navy, premium blue, soft slate, success green, warning amber, error red
- Motion: subtle and purposeful, with respect for user focus

## Reusable Component Guidelines

- Keep components small and composable
- Prefer controlled props and explicit states
- Design for accessibility and localization from the start
- Document visual variants and behavior clearly
