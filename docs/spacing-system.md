# Spacing System

## Scale

- 0: 0px
- 1: 4px
- 2: 8px
- 3: 12px
- 4: 16px
- 5: 20px
- 6: 24px
- 8: 32px
- 10: 40px
- 12: 48px

## Rules

- Use spacing tokens instead of arbitrary values
- Keep related UI elements grouped with 8px or 12px spacing
- Increase spacing for section separation and elevated surfaces
